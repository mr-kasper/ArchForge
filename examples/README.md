# Examples

This directory contains example outputs generated by ArchForge.

## Quick Generation Examples

### React + Clean Architecture + Tailwind v4 + Zustand

```bash
archforge init --name my-react-app --stack react --architecture clean
```

_Interactive prompts will ask for CSS framework (Tailwind v4), state management (Zustand 5), validation (Zod 4), etc._

### React + Feature-Sliced Design

```bash
archforge init --name my-saas --stack react --architecture feature-sliced
```

### Java + Clean Architecture + PostgreSQL

```bash
archforge init --name my-api --stack java --architecture clean --database postgresql --auth jwt
```

### Java + Hexagonal Architecture

```bash
archforge init --name my-hex-api --stack java --architecture hexagonal
```

### Java + Domain-Driven Design
