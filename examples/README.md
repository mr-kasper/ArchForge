# Examples

This directory contains example outputs generated by ArchForge.

> **Install:** `npm install -g @archforge/cli` or run directly with `npx @archforge/cli init`
>
> **npm:** [@archforge/cli](https://www.npmjs.com/package/@archforge/cli)
>
> **Source:** [github.com/mr-kasper/archforge](https://github.com/mr-kasper/archforge)

## Quick Generation Examples

### React + Clean Architecture + Tailwind v4 + Zustand

```bash
# Using npx (no install needed)
npx @archforge/cli init --name my-react-app --stack react --architecture clean

# Or if installed globally
archforge init --name my-react-app --stack react --architecture clean
```

_Interactive prompts will ask for CSS framework (Tailwind v4), state management (Zustand 5), validation (Zod 4), etc._
