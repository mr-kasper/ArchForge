# Examples

This directory contains example outputs generated by ArchForge.

## Quick Generation Examples

### React + Clean Architecture

```bash
archforge init --name my-react-app --stack react --architecture clean --database postgresql --auth jwt
```

### React + Feature-based

```bash
archforge init --name my-saas --stack react --architecture feature-based --database none --auth jwt
```

### Java + Clean Architecture

```bash
archforge init --name my-api --stack java --architecture clean --database postgresql --auth jwt
```

### .NET + Clean Architecture

```bash
archforge init --name my-service --stack dotnet --architecture clean --database postgresql --auth jwt
```
